# Christmas Tree Timer

## Basic Description

Make a simple controller to power an LED pre-lit Christmas Tree.  In addition, also supply power to SAO Ornament boards

## Requirements

1. Current Tree requires approx 28VDC @ 0.2A.
2. Changes from colors to white light by swapping polarity.
3. Has a built in timer.
4. Provides downstream power.  Ideally 28VDC to be downconverted by POL converters
5. Some creature comforts (debugging, setting time, maybe Wifi)
